STEP 1

How to set up your Hardware?

Start the Node MCU

It’ll try to connect to a previously connected wifi, if it fails to connect, due to not finding a wifi with those credentials, then it’ll go into access point mode, and turn into an access point.

Wifi details

Once the Node MCU goes into access point mode, it’ll be shown as a wifi called ‘jatayu’.

Connect

Use any device to connect to how2electronics.

Completion of Process

Once connected, open a browser, and enter 192.168.4.1 in the address bar, and then fill the required details. This will complete the configuration process.

STEP 2

How the Software works?

1. Register

Register as an organization in the website

2. Login

Login to the Website

3. Create

Choose the Create Geofence option

4. Upload Image

Add the image of the indoor map of the floor in which you want to create the geofence along with the floor name.

5. Add AP

After you’ll be redirected to the create geofence section, choose add AP and enter the ssid and mp of the wifi points, and mark the points on the map.

6. Draw the Geofence

Hit add AP to add the points, then draw the geofence on the map and choose pass the coordinates to save the geofence.

7. Live Tracking

To check the created geofences, go to show all maps section and choose the live tracking button in the show all geofence section to see the live location of the patients.

8. Notification

Notifications will pop up when there is a breach in the geofence.
